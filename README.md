# BashGitConfig
skrip bash untuk konfigurasi Git di terminal. 

menghasilkan kunci SSH dan menguji koneksi ke GitHub. 

# Instalasi

```
git clone https://github.com/1amkaizen/BashGitConfig/
```

```
cd BashGitConfig
```

```
./bashgitconfig
```

Kemudian masukkan username dan email github kalian.
